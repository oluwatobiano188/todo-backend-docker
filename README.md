Todo Backend API (Dockerized)

Production-ready REST API for managing Todos.
Built with Node.js, Express, and MongoDB, fully containerized using Docker with multi-stage builds and environment-based configuration.

This project simulates a real-world backend deployment setup and focuses on container best practices.

Tech Stack

Node.js

Express.js

MongoDB

Docker

Docker Compose

Features

RESTful API (CRUD operations)

MongoDB integration

Multi-stage Docker build

Environment variable configuration

Docker networking (App + Database)

Optimized production image

Health check support
